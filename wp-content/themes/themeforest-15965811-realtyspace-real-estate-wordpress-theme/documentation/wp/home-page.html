Home page sections configuration¶

To edit the home page, open go to Pages / Front Page / Edit

You can setup any Visual Composer element, or add a custom Realtyspace sections. You can:

  1. Sort sections with Drag&Drop
  2. Edit sections Row Settings
  3. Edit Section Settings
  4. Setup order and visibility of setcion’s items
  5. Disable breadcrumbs options
  6. Set a custom template

Tip

If you created a new home page, to enable it do to Customizer / Static Front Page / Front Page Select / Choose yours page / Save Changes

Troubleshooting¶

How to center 2 agents on home page?¶

Add code in Appearance / Customize / Custom CSS&JS / Add css :

.listing.worker--index{
    display: flex;
    justify-content: center;
}

Slider Revolution search box.¶

The search box can be edited from Slider Revolution editor. To do that open Slider Revolution tab, and follow these steps:

  1. Click on the “Realtyspace Slider11”.

  2. Click “Static/Global Layers”.

  3. Open Layers List.

  4. Click on the “pen” to edit layer. Here is placed shortcode: [cf47rs_property_search_bar  fields="street, contract_type, property_feature, property_type, price, area"].

  5. Update field="" attribute depending on your needs. Here are fields available for use:

    • sku
    • agents
    • area
    • land_area
    • bathroom
    • bedroom
    • contract_type
    • description
    • street
    • garage
    • limit
    • location
    • mode
    • price
    • property_feature
    • property_type
    • sort
    • year_built
