Sidebar zones
-------------

* Sidebar on supbages
* 3 footer columns

If you have imported demo content, there is added 5 specific sidebars

* Sidebar Properties & Agent - on property details page, which include agent widget
* Sidebar Blog - on blog listing/details pages
* Sidebar Agents - on agents listing/details pages
* Sidebar Properties - on properties listing/details pages
* Sidebar IDX Press  - on IDX Press listing/details pages
