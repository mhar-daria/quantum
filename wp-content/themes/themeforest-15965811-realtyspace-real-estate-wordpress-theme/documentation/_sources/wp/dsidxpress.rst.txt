dsIDXpress integration
----------------------

First you need download and activate "dsIDXPress" plugin, then you will get new VC section, widgets, and listings pages.

To add search form on home page, follow these steps:

#. Create new row in VC
#. Insert **"Property Hero"** section
#. Check field "Show custom shortcode"
#. Insert in next field shortcode ``[idx-quick-search format="vertical"]``
#. Save
