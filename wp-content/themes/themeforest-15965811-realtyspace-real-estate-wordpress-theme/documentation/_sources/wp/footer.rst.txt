Edit footer widgets
+++++++++++++++++++

Widgets can be customized in **Appearance / Widgets** (Footer columns)

.. image:: ../img/footer-edit.png

Change copyright text
+++++++++++++++++++++

Go to Customize / Footer / Copyright text. Copyright section also support HTML tags.


Footer background and styles
++++++++++++++++++++++++++++

Go to Cusomize / Footer / Use custom footer background? (Yes) / Background image


Disable scrollup button
+++++++++++++++++++++++

Go to Customize / Layout / Enable scrollup button / Off
